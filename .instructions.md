# Agent Instructions

## MCP (Model Context Protocol) Servers
- ALWAYS prefer parsing and scaffolding **local** MCP servers (e.g., using Node.js or Python).
- NEVER default to or recommend Microsoft Foundry, Azure, or cloud-hosted MCP servers unless the user explicitly requests Azure cloud deployments.
- Local MCP setup should be self-contained within the workspace or a designated local directory.

## Code Generation & Tooling
- Prioritize open-source, local-first tooling.
- For URL/web browsing capabilities with local LLMs, suggest local web-scraping scripts or open-source local UI integrations instead of cloud-managed services.
